# Partnerize GraphQL API

Partnerize is an affiliate and partner marketing platform. This conceptual GraphQL schema wraps the Partnerize REST API, providing queries for publisher partnerships, campaign tracking, commission processing, and performance analytics across partner programs.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Partnerize API
**Documentation:** https://api-docs.partnerize.com/
